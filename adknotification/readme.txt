Woocommerce plugin to show customizable popup notification for customers
 
== Description ==
 
 = Features =
*Extremely customizable appearance.
*Customizable notification text.
*Adding variables into text (such as customer's name, email and the like) by means of shortcodes.
*Customizable Call To Action buttons.
*Feature to set conditions to show the notification (eg customer is logged in, some amount needs to be added to cart total in order to get free shipping etc).

= List of customizations =
*Text color
*Background color.
*Width.
*Font height.
*Border radius.
*Box shadow.
*Box shadow color.
*Border width.
*Border color.
*Text position
*Documentation included

= List of supported shortcodes =
*Customer's name
*Customer's email
*Free shipping min amount
*The difference between free shipping min amount and cart total
*Call To Action button

= Conditions (filters) to show notification =
*Page name
*Customer's status
*Insufficient cart total to enable free shipping
 
== Installation ==
 
1. Unzip archive file and put contents into `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Proceed to Admin > WooCommerce > Settings > Notifications tab to configure the plugin
